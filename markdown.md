# Bilingual Blog / Двуязычный блог

A bilingual blog template supporting English and Russian languages, designed to be hosted on GitHub Pages.

## Features
- 🌐 Full bilingual support (English/Russian)
- 🔄 Easy language switching
- 📱 Responsive design
- 💾 Saves language preference
- 🎨 Clean, modern design

## How to Use
1. Fork this repository
2. Enable GitHub Pages in repository settings
3. Customize the content in `index.html`
4. Add more blog posts following the same pattern
5. Your blog will be live at `https://[your-username].github.io/[repository-name]`

## Customization
- Edit `style.css` to change the appearance
- Add more languages by adding `data-[language]` attributes
- Modify `script.js` to add more functionality
